# Getting Started

### RocketMQ ###
## 1. cmd : mqnamesrv  (do not close the window)
## 2. another cmd : mqbroker -n 127.0.0.1:9876 autoCreateTopicEnable=true  (do not close the window)

### Redis ###
## 1. cmd : redis-server

### jmeter ###
## 1. directly open jmeter.bat under jmeter/bin folder
